# IdeaVault Server

Express API scaffold for the IdeaVault startup idea sharing platform.

## Features

- JWT token generation for email/password and Google login flows.
- Protected idea create, update, and delete endpoints.
- Protected comment create, update, and delete endpoints.
- Search ideas by title and filter by category.
- Ready for MongoDB persistence when database credentials are added.

## Run Locally

```bash
npm install
npm run dev
```

Default URL: `http://localhost:5000`
